Простая База данных

Создайте простую базу данных для хранения книг. В ней создайте таблицу books. Добавьте в неё поля для хранения идентификатора книги, её названия, года выпуска и автора. Подумайте, какой тип данных должен быть у каждого столбца. Где должно быть автоинкрементирование, Какие значения могут быть Null, а какие нет. После чего добавьте в вашу Базу данных несколько книг с помощью встроенных инструментов.

В качестве ответа пришлите ссылку на репозиторий с вашим проектом (конкретно на сайте github.com).
