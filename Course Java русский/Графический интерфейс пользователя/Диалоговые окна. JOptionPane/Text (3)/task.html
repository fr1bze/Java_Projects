Выбор разрешения

Создайте диалоговое окно с выбором разрешения экрана. Реализуйте это с помощью выпадающего списка. После чего встройте своё диалоговое окно в проект. При запуске проекта должно появляться ваше диалоговое окно с выбором разрешения. После чего должно появиться окно соответствующего размера.

В качестве ответа пришлите ссылку на репозиторий с вашим проектом (конкретно на сайте github.com).
