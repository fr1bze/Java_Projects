JSpinner

Создайте приложение, содержащее в себе элемент JSpinner. 

Ваше приложение должно содержать элемент JSpinner, вам нужно указать, сколько книг вы возьмёте с собой в лагерь. Ниже должна находится кнопка "Ответить", при нажатии на которую в надписи "Ответ" должен появляться выбранный вами ответ.

В качестве ответа пришлите ссылку на репозиторий с вашим проектом (конкретно на сайте github.com).
