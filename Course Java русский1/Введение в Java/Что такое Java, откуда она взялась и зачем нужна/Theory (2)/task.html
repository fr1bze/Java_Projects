Критерии прохождения курса

Баллы в этом курсе начисляются за решение задач. Курс открыт без дедлайнов, поэтому вы можете решать задачи в любое время и получать за них баллы. 

Количество попыток не ограничено.

Ваш общий прогресс отображается в оглавлении курса.

Критерии для получения сертификата таковы:

  • обычный сертификат выдается за 99 баллов;
  • сертификат с отличием — за 119 баллов и более.
